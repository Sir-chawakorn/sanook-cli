# Correction Pairs

> เจอ failure → จดคู่ "ผิด → ถูก" + บทเรียน เพื่อไม่ผิดซ้ำ (feed เข้า Playbooks/Distillations)

## Pairs

_(รูปแบบ:)_

```
### YYYY-MM-DD — <failure mode>
- ❌ ทำผิด: ...
- ✅ ที่ถูก: ...
- บทเรียน: ... (→ promote เป็น Playbook/Distillation ถ้าเจอ ≥3 ครั้ง)
```
